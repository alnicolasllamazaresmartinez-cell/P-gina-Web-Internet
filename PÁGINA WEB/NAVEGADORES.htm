Un navegador web es una aplicación que permite acceder y visualizar páginas de Internet, interpretando contenidos como HTML, CSS y JavaScript para facilitar la interacción del usuario.

Los navegadores surgieron en 1990 con WorldWideWeb, creado por Tim Berners-Lee. Con el tiempo aparecieron navegadores populares como Mosaic, Netscape, Internet Explorer y Firefox. Actualmente, los más utilizados son Google Chrome, Safari y Microsoft Edge.

El navegador funciona al traducir una URL mediante DNS, conectarse a un servidor con HTTP/HTTPS, descargar los archivos de la página y mostrarlos en pantalla, permitiendo además la interacción del usuario.

Entre los principales navegadores destacan Chrome, Firefox, Safari, Edge y Opera, cada uno con diferentes ventajas según velocidad, seguridad y funciones.
